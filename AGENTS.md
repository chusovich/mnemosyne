# AGENTS.md — mnemosyne

NixOS system flake + Docker Compose services for a self-hosted media server.

## Repo structure

- `configuration.nix` — NixOS config for host `mnemosyne`.
- `hardware-configuration.nix` — generated by `nixos-generate-config`; don't hand-edit.
- `flake.nix` — pinned to `nixos-26.05`, `x86_64-linux`, imports `nix-core.serverModules.default` (common server/Docker settings live in that flake).
- `backups.nix` — Restic backup jobs, **not imported** by `configuration.nix` and still has placeholders (`<name>`, `<path>`).
- `containers/` — **git submodule** (`chusovich/containers.git`) with reusable Compose bases. Don't edit these files here; change upstream and bump the submodule.
- Top-level service directories — `traefik/`, `immich/`, `jellyfin/`, `homebox/`, `radicale/`, `beszel/` — each has a `compose.yaml`. Most extend the templates in `containers/`.

## Common commands (via `just`)

```bash
just --list          # show recipes
just switch          # build and activate NixOS config
just test            # activate temporarily, no bootloader entry
just dry             # preview what would change
just build           # build without switching
just fmt             # run treefmt (nixpkgs-fmt, prettier, yamlfmt)
just check           # nix flake check
just update          # nix flake update
just update-input X  # update a single flake input
```

`flake.nix` defines the formatter with `treefmt-nix`. Excludes from formatting: `flake.lock`, `*.env`, `containers/**`.

## Docker services

- Start Traefik before dependent services:
  ```bash
  cd traefik && docker compose --profile proxy up -d
  ```
- Most web services attach to the external Docker network `traefik` and rely on Traefik for TLS.
- Traefik expects the Cloudflare DNS API token at `/home/mnemosyne/secrets/CF_DNS_API_TOKEN` (bind-mounted). `secrets/` is gitignored; never commit it.
- Certificates are stored at `/home/calebh/certs`.
- Local service hostnames use `*.apps.husovich.com`; the shared `containers/traefik.yaml` still references `*.subdomain.husovich.com`.
- Immich is configured by `immich/.env` (tracked) and served through Traefik at `immich.apps.husovich.com`.
- Jellyfin exposes ports `8096/tcp` and `7359/udp` directly.

## Storage / host assumptions

- ZFS pool `media-pool` is imported on boot; hostId is set for ZFS.
- Static network config: `192.168.10.10/24`, gateway/DNS `192.168.10.1`.
- Service data paths: `/home/mnemosyne/app-data/<service>` and `/media-pool/<service>`.

## Backups

`backups.nix` is present but inactive. To enable:

1. Add `./backups.nix` to `configuration.nix` imports.
2. Replace every `<name>` and `<path>` placeholder with real values.

## Secrets

- Do not commit `secrets/` or real API tokens/passwords.
- `immich/.env` is tracked and currently contains the default DB password `postgres`; rotate before production.

## Agent helpers

- `opencode.json` configures the `mcp-nixos` MCP server; use it for nixpkgs / NixOS option queries.
